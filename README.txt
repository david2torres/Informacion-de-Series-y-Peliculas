Informacion de Series Y Peliculas.

--Para iniciar este aplicativo, abra su editor de codigo (Atom o SublimeText) o el que tenga por preferencia.
--Cree una carpeta llamada sfp, y guarde todos los archivos del repositorio Informacion de Series Y Peliculas en ella.
--Ingrese a la consola de comandos, y dirijase a la ruta donde haya guardado este proyecto, Asegurece de estar en la carpeta (sfp).
--Una vez este en esa ubicación, ejecute el comando ng serve -o.
--Automaticamente se abrira su navegador con la aplicacion 

--Si llega a presentar fallas con el puerto, puede cambiarlo con el comando ng serve -o --port 4201
